# Зачарователь

Local mod for Melvor Idle.

Добавляет новую профу/страницу `Зачарователь`, слот снаряжения `Чарм` и предмет `Чарм-кристалл`.

Предметы с passive/modifier бонусами можно приносить в жертву, чтобы добавить их modifiers на Чарм-кристалл. Повторные жертвы складываются.

## Install

Pack the contents of this folder into a zip archive so `manifest.json` is at the archive root, then add it as a local mod through the Melvor Idle Creator Toolkit or upload it to mod.io.

## Notes

- Цена первой жертвы: 1 000 золота.
- Каждая следующая цена умножается на 1.5.
- Максимальная цена: 100 000 000 золота.
- Боевые характеристики предмета не копируются, только `modifiers`.
